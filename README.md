## 설치 방법

OS X & 리눅스:
1. Clone the repo
```sh
git clone https://github.com/JUNGganzi/nodejs-972dog.git
```
2.  Install NPM packages 
```
npm init
```
3. install dependencies Modules
```
npm install
```

## 현재 상황
- 현재 개발 진행상황

	원하는 의류항목 사이즈 체크
